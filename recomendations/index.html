    Software que utilizo

    Esto es una lista del software libre y de código abierto que utilizo:

    La lista puede variar con el tiempo, por lo que tal vez encuentres software que te sorprenda o puede que no lo encuentres más. Esta lista también es útil para hacer referencias a software apropiado que puede serte útil en el día a día.

    Procesado de Imagen y Biomedicina

    • 3D Slicer es un software gratuito y de código abierto para la visualización, procesamiento, segmentación, registro y análisis de imágenes y mallas médicas, biomédicas y otras imágenes 3D; y la planificación y navegación de procedimientos guiados por imágenes.

    • Weasis DICOM medical viewer es un visor DICOM multipropósito, autónomo y basado en web, con una arquitectura altamente modular. Es un visor clínico muy popular utilizado en la asistencia sanitaria por hospitales, redes sanitarias, ensayos de investigación multicéntricos y pacientes.

    Edición de fotografía

    • Darktable es una aplicación de flujo de trabajo fotográfico de código abierto y desarrollador de RAW. Una mesa de luz y cuarto oscuro virtual para fotógrafos.

    Programación

    • //grep.app busca código en más de medio millón de repositorios públicos en GitHub.

    Deporte

    • Wger es un proyecto colaborativo de código abierto, creado por entusiastas del fitness de todo el mundo.

    Alimentación

    • grocy es una solución web para la gestión de la compra y el hogar.
